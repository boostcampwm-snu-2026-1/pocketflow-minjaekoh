# Pocketflow

Pocketflow는 AI를 활용한 개인 재무 대시보드 프로젝트입니다.

## 프로젝트 범위
- 현금흐름 예측 대시보드
- 지출 관리 및 분석
- 고정비 / 준고정비 설정
- AI 기반 구매 및 소비 분석

## 현재 문서
- `PRD.md`: 프로젝트 기획서 초안
- `issues_backlog.md`: 작업 이슈 목록
- `WORKFLOW.md`: 개발 워크플로우 및 문서 규칙
- `FOLDER_STRUCTURE.md`: 폴더 구조 설계

## 운영 원칙
- 작업 관리는 GitHub Issues로 진행합니다.
- 문서와 워크플로우는 UTF-8로 저장합니다.
- Vercel AI로 생성한 화면 코드는 최종본이 아니라 참고용으로만 사용합니다.
