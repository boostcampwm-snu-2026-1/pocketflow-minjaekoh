# Vercel AI 참고 코드

이 폴더는 Vercel AI로 생성한 화면 코드를 참고하기 위한 영역입니다.

## 보관 기준
- 화면 구조를 파악하는 데 필요한 페이지와 직접 작성된 컴포넌트만 보관합니다.
- `pnpm-lock.yaml`, placeholder 이미지, shadcn/ui 전체 생성 파일은 커밋 크기를 줄이기 위해 제외합니다.
- 이 코드는 실제 앱 코드가 아니며, 필요한 부분만 `src` 구조에 맞게 다시 옮깁니다.
